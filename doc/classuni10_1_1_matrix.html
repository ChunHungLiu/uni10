Detailed Description

The Matrix class defines a common matrix.

Matrix is an auxilliary class used to extract block information on UniTensor and perform linear algebra operations. A symmetric tensor contains Block's with corresponding Qnum's. Each block is a Matrix with tensor elements.

The Matrix follows the C convention that the memory storage is row-major and indices start from 0.

Examples:
egM1.cpp, egM2.cpp, and egU2.cpp.
