Detailed Description

The UniTensor class defines the symmetric tensors.

A UniTensor consists of Bond's carrying quantum numbers Qnum's. The tensor elements are organized as quantum number blocks. The Qnum's on the Bonds defines the size of the Qnum blocks and the rank of UniTensor is defined by the number of Bond's.

Each Bond carries a label. Labels are used to manipulate tensors, such as in permute, partialTrace and contraction.
Operations on tensor elements is pefromed through getBlock and putBlock functions to take out/put in block elements out as a Matrix.
See also
Qnum, Bond, Matrix
Examples:
egN1.cpp, egU1.cpp, egU2.cpp, and egU3.cpp.

Driver function for High-order SVD.

Performs High order SVD of UniTensor.

Parameters
[in]group_labelsOrdered labels of the bonds
[in]groupsNumber of external bonds in each mode
[in]groupSizeNumber of modes
[out]LsSingular values on each direction
***** Example ******
____________ ______ ____
| | 0___| U1 |__-1 ___ __-2___| |___1
8___| |___0 |______| _|___|_ | U2 |___2
| | | | |____|
9___| |___1 __11_| |
| | __3__| |
11__| |___2 | S |
| | ______ | | ____
6___| |___3 8__| |__-4__| |__-3_| |___4
| | 9__| U4 | |_______| | U3 |___5
5___| |___4 |______| |____|___6
| |
|____________|

We want to decompose the tensor into a tensor of four modes and two bonds unchanged.

Mode 1 {0,-1}
Mode 2 {1,2,-2}
Mode 3 {4,5,6,-3}
Mode 4 {8,9,-4}
Fixed {3,11}

The input parameters are

group_labels = { 0, 1, 2, 4, 5, 6, 8, 9, 3, 11 }
groups={1,2,3,2}
groupSize=4

Returns unitaries U1, U2, U3, U4, and core tensor S
